# Glossary

Key terms used throughout this documentation.

## AI & Agents

| Term | Definition |
|------|------------|
| **AI Agent** | An AI system that can use tools, make decisions, and execute tasks autonomously |
| **Skill** | A reusable capability that extends an AI agent's abilities |
| **MCP** | Model Context Protocol — a standard for connecting AI to external tools |
| **Subagent** | A specialized agent spawned to handle specific tasks in isolation |
| **Prompt Engineering** | The practice of designing inputs to get desired outputs from AI |

## Claude & Anthropic

| Term | Definition |
|------|------------|
| **Claude Code** | Anthropic's CLI tool for AI-assisted development |
| **SKILL.md** | The main file defining a Claude Code skill |
| **Task Tool** | Claude Code's mechanism for spawning subagents |
| **Opus/Sonnet/Haiku** | Claude model tiers (most to least capable) |

## Development Tools

| Term | Definition |
|------|------------|
| **Cursor IDE** | AI-first code editor with MCP support |
| **OpenSkills** | Skill loader for AI coding assistants |
| **Gemini CLI** | Google's command-line interface for Gemini |
| **Notion MCP** | MCP server for Notion API integration |

## Web3

| Term | Definition |
|------|------------|
| **Smart Contract** | Self-executing code on a blockchain |
| **DeFi** | Decentralized Finance — financial services on blockchain |
| **NFT** | Non-Fungible Token — unique digital assets |
| **DAO** | Decentralized Autonomous Organization |
| **Wallet** | Software/hardware for managing crypto assets |

## Workflow & Automation

| Term | Definition |
|------|------------|
| **SOP** | Standard Operating Procedure — step-by-step guide |
| **3P Update** | Progress/Plans/Problems format for status updates |
| **Webhook** | HTTP callback for event-driven automation |
| **Bot Token** | Authentication key for bot APIs (e.g., Telegram) |

## File Formats

| Extension | Description |
|-----------|-------------|
| `.md` | Markdown document |
| `.pptx` | PowerPoint presentation |
| `.docx` | Word document |
| `.pdf` | Portable Document Format |
| `.json` | JavaScript Object Notation |
