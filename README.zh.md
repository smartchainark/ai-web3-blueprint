## 写在前面

大家好，我是 **king**，x402 协议贡献者，Cursor 2025 全球排行榜前 4%。

在开发多个 AI x Web3 项目后，我们深刻意识到 **Agent Skills 才是未来的趋势** — AI 不再只是聊天，而是能够调用工具、执行任务、协作完成复杂工作流的智能体。

这份蓝皮书记录了我们在实践中总结的 SOP、技巧和思考，聚焦于 **AI Agent 与 Web3 的交叉领域**。
