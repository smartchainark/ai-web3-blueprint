> AI 与 Web3 交叉领域的 SOP、指南和洞察合集。

## 欢迎

我是 **king**，x402 协议贡献者，Cursor 2025 全球排行榜前 4%。

在开发多个 AI x Web3 项目后，我们意识到 **Agent Skills 是未来** — AI 不再只是聊天，而是能够调用工具、执行任务、协作完成复杂工作流的智能代理。

本蓝皮书记录了我们从实践中总结的 SOP、技术和洞察，聚焦于 **AI Agent 与 Web3 的交叉领域**。

## 你将学到

- **AI Agent 工作流** — 集成 Claude、Gemini、Notion 等 AI 工具的实用 SOP
- **技能开发** — 如何跨平台构建、部署和管理 AI Agent 技能
- **Web3 集成** — 将 AI Agent 与区块链和去中心化系统连接
- **最佳实践** — 来自真实实现的模式和反模式

## 涵盖主题

- **AI Agents** — 架构、工具使用、多代理系统
- **技能系统** — Claude Code Skills、OpenSkills、MCP 服务器
- **自动化** — Notion 集成、文档导出、通知
- **Web3 开发** — 智能合约、DeFi、NFT（即将推出）
