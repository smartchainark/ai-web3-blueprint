# 术语表

本文档中使用的关键术语。

## AI 与代理

| 术语 | 定义 |
|------|------|
| **AI Agent** | 能够使用工具、做出决策并自主执行任务的 AI 系统 |
| **Skill（技能）** | 扩展 AI 代理能力的可复用功能 |
| **MCP** | Model Context Protocol — 连接 AI 与外部工具的标准协议 |
| **Subagent（子代理）** | 为隔离处理特定任务而生成的专用代理 |
| **Prompt Engineering** | 设计输入以获得 AI 期望输出的实践 |

## Claude 与 Anthropic

| 术语 | 定义 |
|------|------|
| **Claude Code** | Anthropic 的 AI 辅助开发 CLI 工具 |
| **SKILL.md** | 定义 Claude Code 技能的主文件 |
| **Task Tool** | Claude Code 生成子代理的机制 |
| **Opus/Sonnet/Haiku** | Claude 模型层级（从强到弱） |

## 开发工具

| 术语 | 定义 |
|------|------|
| **Cursor IDE** | 支持 MCP 的 AI 优先代码编辑器 |
| **OpenSkills** | AI 编码助手的技能加载器 |
| **Gemini CLI** | Google 的 Gemini 命令行界面 |
| **Notion MCP** | 用于 Notion API 集成的 MCP 服务器 |

## Web3

| 术语 | 定义 |
|------|------|
| **Smart Contract（智能合约）** | 在区块链上自动执行的代码 |
| **DeFi** | 去中心化金融 — 基于区块链的金融服务 |
| **NFT** | 非同质化代币 — 独特的数字资产 |
| **DAO** | 去中心化自治组织 |
| **Wallet（钱包）** | 管理加密资产的软件/硬件 |

## 工作流与自动化

| 术语 | 定义 |
|------|------|
| **SOP** | 标准操作流程 — 分步指南 |
| **3P 更新** | Progress/Plans/Problems 格式的状态更新 |
| **Webhook** | 用于事件驱动自动化的 HTTP 回调 |
| **Bot Token** | Bot API 的认证密钥（如 Telegram） |

## 文件格式

| 扩展名 | 描述 |
|--------|------|
| `.md` | Markdown 文档 |
| `.pptx` | PowerPoint 演示文稿 |
| `.docx` | Word 文档 |
| `.pdf` | 便携式文档格式 |
| `.json` | JavaScript 对象表示法 |
