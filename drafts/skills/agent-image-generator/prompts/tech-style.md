# 科技风格模板 (Tech Style)

适用于技术博客、产品页面、开发者社区配图。

## 基础提示词

```yaml
name: tech-style
category: 科技/技术
description: 现代科技感，适合技术内容配图

base_prompt: |
  A modern tech-inspired illustration of {subject},
  featuring clean geometric shapes, circuit board patterns,
  and glowing neon accents on a dark gradient background.
  Style: {style}, minimalist, professional.
  Color scheme: {color_scheme}.
  Lighting: soft ambient glow with subtle highlights.
  Composition: balanced, with clear focal point.
  Quality: high resolution, crisp details, 4K.
```

## 风格变体

### 1. 深空科技 (Dark Space Tech)
```
深蓝色背景，星空元素，霓虹蓝色光效，
未来感十足，适合 AI/ML 相关主题
```

**修饰词**: `deep space aesthetic, floating particles, blue neon glow`

### 2. 简约线条 (Minimal Lines)
```
白色或浅灰背景，细线条图形，
简洁现代，适合工具/流程类主题
```

**修饰词**: `minimal line art, white background, thin geometric lines`

### 3. 渐变抽象 (Gradient Abstract)
```
紫蓝渐变，抽象几何形状，
动感现代，适合产品发布/新功能
```

**修饰词**: `vibrant gradient, abstract shapes, purple to blue transition`

### 4. 电路纹理 (Circuit Pattern)
```
电路板纹理背景，数据流动效果，
技术氛围浓厚，适合系统架构/基础设施
```

**修饰词**: `circuit board texture, data flow visualization, tech infrastructure`
