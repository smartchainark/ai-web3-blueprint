# Agent 智能生图 (Agent Image Generator)

通过 Claude 推理能力 + 预制提示词模板，生成精准的图片描述，然后调用 Gemini 官网免费生图，自动下载到当前工作目录。

## 核心优势

| 特性 | 说明 |
|------|------|
| **完全免费** | 使用 Gemini 官网生图，无 API 费用 |
| **智能提示词** | Claude 推理 + 预制模板，生成专业级描述 |
| **自动下载** | 图片自动保存到当前目录 |
| **网站配图友好** | 预设多种网站配图场景 |

## 工作流程

```
┌─────────────────────────────────────────────────────────────┐
│                    Agent 智能生图流程                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1️⃣ 用户描述需求                                             │
│     "帮我生成一张关于AI自动化的科技风配图"                      │
│                         ↓                                   │
│  2️⃣ Claude 推理 + 预制模板                                   │
│     ┌─────────────────────────────────────────────┐        │
│     │ 分析需求 → 选择模板 → 融合上下文 → 生成提示词  │        │
│     └─────────────────────────────────────────────┘        │
│                         ↓                                   │
│  3️⃣ 打开 Gemini 官网                                         │
│     ┌─────────────────────────────────────────────┐        │
│     │ Chrome DevTools → gemini.google.com         │        │
│     │ 自动输入提示词 → 等待生成                     │        │
│     └─────────────────────────────────────────────┘        │
│                         ↓                                   │
│  4️⃣ 自动下载图片                                             │
│     ┌─────────────────────────────────────────────┐        │
│     │ 检测生成完成 → 右键保存 → 当前目录            │        │
│     │ 文件命名: {timestamp}_{description}.png     │        │
│     └─────────────────────────────────────────────┘        │
│                         ↓                                   │
│  5️⃣ 返回结果                                                │
│     "图片已保存: ./images/20251229_ai_automation.png"       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 使用方式

### 基础用法

```
# 简单描述
生成一张科技风的配图

# 详细需求
帮我画一张：程序员在使用AI工具编程的场景，蓝色科技风，16:9横版

# 指定场景
生成网站 Hero 区域的配图，主题是自动化工作流
```

### 高级用法

```
# 结合文章上下文
为这篇关于 MCP Playwright 的文章生成配图

# 批量生成
为文章的 3 个金句各生成一张配图

# 指定风格
使用扁平插画风格生成一张团队协作的图
```

## 预制提示词模板

详见 [prompts/](./prompts/) 目录中的模板文件。

## 前置要求

- MCP Playwright 已安装
- Google 账号（用于 Gemini）
- 浏览器会话持久化配置

## 输出目录

默认保存到当前工作目录的 `./images/` 文件夹：

```
./images/
├── 20251229_143052_ai_automation.png
├── 20251229_143215_team_collaboration.png
└── 20251229_143330_workflow_diagram.png
```

## 注意事项

- Gemini 生图为免费服务，可能有速率限制
- 首次使用需登录 Google 账号
- 生成时间约 10-30 秒
