## About

This repository contains in-depth guides, tutorials, and insights on building at the intersection of AI and Web3:

- **AI Agents** - Autonomous agents, tool use, and multi-agent systems
- **Web3 Development** - Smart contracts, DeFi, NFTs, and decentralized applications
- **AI x Web3** - On-chain AI, decentralized inference, tokenized models, and more
